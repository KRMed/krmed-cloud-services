# Resume Builder

> **Status: Planned**

A personal resume generation tool backed by a database of my own experiences, bullet points, and achievements. Given a job description or role, it builds a tailored resume ready to download no reformatting, no copy-pasting, just apply.

## What it will do

- Maintain a structured database of experiences, skills, and bullet points
- Accept a job description or prompt as input
- Select and arrange the most relevant content for that role
- Output a polished, download-ready resume

## Why

Tailoring a resume for every application is time-consuming and error-prone. This centralizes everything I've done in one place and automates the per-application customization, making it faster to apply well rather than apply broadly.
